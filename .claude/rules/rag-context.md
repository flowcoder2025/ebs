# RAG Context — 주제별 파일 매핑

> 작업 시작 전 관련 RAG 파일을 로드하여 컨텍스트를 확보합니다.

## 매핑

| 주제 | RAG 파일 | 로드 시점 |
|------|---------|----------|
| 전체 진행 상태, WI 이력 | `00-timeline.md` | 세션 시작, 상태 확인 |
| 캐릭터 프롬프트, 스타일가이드, 색상 | `01-characters.md` | 캐릭터 관련 작업 |
| 씬 배경, 에피소드 구성, 카메라 | `02-scenes.md` | 씬/배경 관련 작업 |
| ComfyUI, Runway, 제작 도구 | `03-pipeline.md` | 이미지/영상 생성 작업 |
| 공모 서류, 마감, 제출 | `04-submission.md` | 서류/제출 관련 작업 |
| 의사결정 이력, 맥락 | `decisions-log.md` | 이전 결정 참조 필요 시 |

## 규칙
- API, 페이지, 스키마 변경 시 해당 RAG 파일 즉시 업데이트
- 새 WI 완료 시 `00-timeline.md` 업데이트 필수
- 주요 결정 시 `decisions-log.md`에 기록
